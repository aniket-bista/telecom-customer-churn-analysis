# telecom-customer-churn-analysis
Telecom Customer Churn Analysis (R)

This project analyzes a telecom customer churn dataset using R as part of a Business Analytics final project.

Key steps:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Predictive modeling using logistic regression
- Model evaluation using accuracy, ROC/AUC, and confusion matrix

Tools: R, caret, ROCR
